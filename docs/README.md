## 🔄 技術スタック

### フロントエンド

- **HTML5**: セマンティックマークアップ
- **CSS3**: Tailwind CSS + カスタムスタイル
- **JavaScript (ES6+)**: モジュラーアーキテクチャ
- **PDF.js**: PDF処理・プレビュー機能

### 外部サービス

- **Azure OpenAI**: GPT-4.1-mini による対話生成
- **CDN**: Tailwind CSS, PDF.js の配信

- 🇰🇷 한국어

## 📊 実装済み機能概要

### コア機能

- **Azure OpenAI 連携**: GPT-4.1-mini による高品質な対話生成
- **PDF 解析**: 決算資料・株主総会資料の自動テキスト抽出
- **質問候補生成**: PDF内容を解析した6つの株主質問候補の自動生成
- **自動対話進行**: 株主と取締役の5ターン対話シミュレーション
- **多言語対応**: 7言語での対話生成とローカライゼーション

### UI/UX機能

- **レスポンシブデザイン**: PC・タブレット・スマートフォン対応
- **言語セレクタ同期**: 2箇所の言語選択ドロップダウンの完全同期
- **質問候補表示**: カード形式での質問候補表示とクリック選択
- **折り畳み設定**: 接続設定・質問候補エリアの自動折り畳み
- **ポップアップモーダル**: 対話内容の詳細表示
- **自動拡張**: 対話表示エリアの動的サイズ調整

### 技術機能

- **ローカルストレージ**: 設定情報の永続化
- **エラーハンドリング**: 包括的なエラー処理とユーザーフィードバック
- **パフォーマンス最適化**: API呼び出し最適化とメモリ管理
- **セキュリティ**: API キーの安全な管理と通信暗号化
